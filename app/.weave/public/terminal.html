    Vim quick reference

    :Ex
    Open the built-in file explorer (netrw) — use this instead of a file browser to navigate and open files right inside the terminal. :Ex . opens the current dir; Enter opens a file, - goes up a level.

    Files & windows

    • :e fileopen / edit a file
    • :wsave (write)
    • :wq / ZZsave & quit
    • :q / :q!quit / force quit
    • :sp / :vssplit horizontal / vertical
    • Ctrl-w wjump between splits

    Editing

    • i / ainsert / append
    • Escback to normal mode
    • dddelete line
    • yy / pyank line / paste
    • u / Ctrl-rundo / redo

    Move & search

    • gg / Gtop / bottom of file
    • 0 / $start / end of line
    • /text then nsearch, next match
    • :%s/old/new/greplace all

    Search all files (VSCode ⌘⇧F)

    • :vimgrep /text/ **/*recursive project search
    • :copenopen results list
    • :cn / :cpnext / prev match
    • :grep -r text .same, via external grep
