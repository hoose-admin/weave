    Vim quick reference

    :Ex
    Open the built-in file explorer (netrw) — use this instead of a file browser to navigate and open files right inside the terminal. :Ex . opens the current dir; Enter opens a file, - goes up a level.

    Files & windows

    • :e fileopen / edit a file
    • :wsave (write)
    • :wq / ZZsave & quit
    • :q / :q!quit / force quit
    • :sp / :vssplit horizontal / vertical
    • Ctrl-w wjump between splits

    Editing

    • i / ainsert / append
    • Escback to normal mode
    • dddelete line
    • dw / dWdelete word / WORD
    • dEdelete to end of WORD
    • yy / pyank line / paste
    • u / Ctrl-rundo / redo

    Move ↕ vertical

    • gg / Gtop / bottom of file
    • Ctrl-d / Ctrl-uhalf-page down / up
    • Ctrl-f / Ctrl-bfull page down / up
    • { / }prev / next paragraph

    Move ↔ horizontal

    • 0 / $start / end of line
    • ^first non-blank char
    • w / bnext / prev word
    • eend of word
    • %jump to matching ( ) [ ] { }

    Search & replace

    • /text then nsearch, next match
    • :%s/old/new/greplace all

    Claude slash commands

    /
