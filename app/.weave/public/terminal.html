    Neovim quick reference

    :Neotree
    Open the neo-tree file browser to explore and open files without leaving the editor. \e toggles it (\ is the leader key). Inside the tree: Enter opens a file, a / d / r add / delete / rename, H toggles hidden files, ? shows every key.
    \gd
    See changes right in the file like GitHub — green added / changed lines, red removed lines shown inline. \gd toggles the inline diff; \gp expands just the hunk under the cursor; \gD opens a full split diff against the last commit.

    Files & windows

    • :e fileopen / edit a file
    • :wsave (write)
    • :wq / ZZsave & quit
    • :q / :q!quit / force quit
    • :sp / :vssplit horizontal / vertical
    • Ctrl-w wjump between splits

    Editing

    • i / ainsert / append
    • Escback to normal mode
    • dddelete line
    • dw / dWdelete word / WORD
    • dEdelete to end of WORD
    • yy / pyank line / paste
    • u / Ctrl-rundo / redo

    Move ↕ vertical

    • gg / Gtop / bottom of file
    • Ctrl-d / Ctrl-uhalf-page down / up
    • Ctrl-f / Ctrl-bfull page down / up
    • { / }prev / next paragraph

    Move ↔ horizontal

    • 0 / $start / end of line
    • ^first non-blank char
    • w / bnext / prev word
    • eend of word
    • %jump to matching ( ) [ ] { }

    Search & replace

    • /text then nsearch, next match
    • :%s/old/new/greplace all

    Git in the buffer

    • \gdtoggle inline diff (red / green)
    • \gppreview this hunk inline
    • ]h / [hnext / prev change
    • \gbblame this line
    • \gDsplit diff vs last commit
